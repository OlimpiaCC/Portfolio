Hotel

Woodland

Woodland is a hotel located in the deep Polish forest. My task was to create a website for the hotel, both in terms of graphic design and front-end.
From the design side, I proposed a color palette closely related to the location of the hotel and the main brand value, which is in close contact with nature. Supplementary colors are mainly pure, light shades of gray, accentuating the modern character of the hotel.
In terms of typography, I opted for a calligraphic handcrafted font named "Limber" for headings and a clean, unpretentious "Poppins" font for paragraphs.
As for the front-end, I used html, scss, and java script. The main solutions used on the website are sliders, parallax, video player and reservation system. The calendar was prepared using an external library, which I expanded with additional functionalities. The user can select the dates of stay from the form. Additionally, a functionality has been added based on LocalStorage, which causes the data selected in the calendar to be stored in the computer's cache memory.

SCOPE: UX, UI, Front-end
